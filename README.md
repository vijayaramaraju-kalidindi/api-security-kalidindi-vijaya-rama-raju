# API key exposure risk
Exposing an API key on GitHub allows unauthorized users to misuse the service, leading to potential financial loss, quota exhaustion, and service disruption. It can also compromise system security if the key is tied to sensitive operations. Public leaks may require key rotation and incident response.

# Privacy policy (city logging)
Logging city names can expose user location data, which may be considered personal information under privacy laws like GDPR. This violates the data minimization principle, which requires collecting only necessary data. Avoiding such logs helps protect user privacy and reduces compliance risks.
